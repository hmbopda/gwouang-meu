04 — Vidéo en direct
Lives, Conférences & Cours

LiveKit Cloud = WebRTC managé, SDK Flutter natif, recording cloud. Zéro infrastructure vidéo à gérer. Gratuit jusqu'à 100 participants simultanés.

🎥 LiveKit Cloud Free tier
Pourquoi LiveKit ? SDK Flutter officiel (livekit_client). WebRTC managé, scalable auto. Recording natif. RTMP vers YouTube/Facebook.

Rooms :
• live_course_{id} — cours de langue
• live_conf_{id} — conférence culturelle
• live_tour_{id} — visite guidée virtuelle
• live_village_{id} — webinaire privé
livekit_client FlutterWebRTCE2E chiffré
📹 Recording & Replay Inclus
Sessions enregistrées → Cloudflare R2. Replay intégré dans la page du cours ou du village.

• Replay cours → language-module
• Replay conférence → page village
• Chapitrage auto via Claude AI
• Transcription + traduction automatique
Cloudflare R2HLS streamingClaude transcription
📡 Diffusion externe RTMP Egress
Diffusion simultanée vers YouTube Live + Facebook Live via RTMP Egress LiveKit. Idéal pour grandes conférences culturelles avec audience diaspora mondiale.

Claude est le cerveau de NGOMALA. Architecture RAG + context engineering + streaming SSE. Prompt caching pour réduire les coûts de 80%.

// 8 CAS D'USAGE CONCRETS
🗺️
Guide Touristique
Sonnet 4.6
Chatbot contextuel au village visité. Répond en FR/EN/ES + langue locale. Connait histoire, rites, gastronomie, itinéraires.
System: "Tu es le guide officiel du village {village_name}. Histoire depuis {year}. Culture: {summary}. Réponds en {user_lang} avec mots en {local_lang}."
~0.01€ / conversation · 500 tokens moy.
🌳
Matching Généalogique
Sonnet 4.6
Analyse l'arbre Neo4j et suggère des liens familiaux manquants. Explique chaque suggestion. Validation humaine obligatoire.
System: "Analyse cet arbre partiel du village {v}. Identifie les liens probables. Score de confiance 0-1. Ne jamais affirmer sans preuve."
~0.05€ / analyse · Opus pour arbres complexes
🗣️
Quiz Linguistiques
Sonnet 4.6
Génère des quiz personnalisés par niveau et dialecte. Corrige avec explications culturelles. Adapte la difficulté en temps réel.
System: "Génère 5 questions niveau {level} en {dialect}. Inclus contexte culturel. Format JSON strict."
~0.003€ / quiz 5 questions
🍲
Enrichissement Culturel
Opus 4.6
Génère et enrichit les fiches culturelles villages. Détecte les connexions avec d'autres peuples africains. Modèle Opus pour la profondeur.
System: "Enrichis la fiche du village {v}. Détecte les liens avec autres cultures. Sources si possible."
~0.15€ / enrichissement · 1×/mois par village
🔗
Connexions Culturelles
Sonnet 4.6
Compare des villages et suggère des liens transversaux : dialectes proches, cuisines similaires, rites partagés, migrations historiques.
System: "Compare ces {n} fiches culturelles. Score de similarité par catégorie (dialecte, cuisine, rites, histoire). Justifie."
~0.08€ / analyse batch villages
🎥
Post-Live Intelligence
Sonnet 4.6
Après chaque live : résumé structuré, chapitrage horodaté, quiz basé sur le contenu, points clés, évaluation étudiants.
System: "Transcription du live '{title}'. Génère: résumé 300 mots, chapitrage timestamps, 5 questions, points clés."
~0.02€ / résumé live 1h
🛡️
Modération Contenu
Sonnet 4.6
Pré-modération publications avant review humaine. Détecte contenus haineux, hors-sujet, spam. Suggère action à l'ambassadeur.
System: "Modérateur culturel NGOMALA. Évalue ce post pour {village}. Retourne JSON: {action, score, reason}"
~0.001€ / post modéré
🏛️
Itinéraires Personnalisés
Sonnet 4.6
Crée des itinéraires selon le profil visiteur (diaspora, chercheur, touriste, famille). Intègre POI, durées, transports, repas locaux.
System: "Visiteur {profile} veut découvrir {village} en {days}j. Budget {budget}. Itinéraire détaillé avec POI culturels."
~0.02€ / itinéraire généré
